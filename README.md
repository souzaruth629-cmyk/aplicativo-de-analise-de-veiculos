# Dashboard de Análise de Veículos

Este projeto é um aplicativo web desenvolvido com Streamlit para análise exploratória de dados de anúncios de vendas de carros. O dashboard permite visualizar e interagir com os dados através de histogramas e gráficos de dispersão.

## Descrição do Projeto

O objetivo deste projeto é criar um aplicativo web interativo que permita aos usuários explorar um conjunto de dados de anúncios de vendas de carros. O aplicativo inclui:

- Análise exploratória de dados (EDA)
- Visualizações interativas
- Interface web responsiva
- Deploy em serviço de nuvem

## Funcionalidades

- **Histogramas**: Visualização da distribuição de variáveis como preço, ano do modelo, quilometragem
- **Gráficos de Dispersão**: Análise de correlações entre diferentes variáveis
- **Controles Interativos**: Botões e caixas de seleção para filtrar e personalizar as visualizações
- **Interface Intuitiva**: Design limpo e fácil navegação

## Tecnologias Utilizadas

- **Python**: Linguagem de programação principal
- **Streamlit**: Framework para desenvolvimento do aplicativo web
- **Pandas**: Manipulação e análise de dados
- **Plotly Express**: Criação de gráficos interativos
- **Render**: Plataforma de deploy em nuvem
